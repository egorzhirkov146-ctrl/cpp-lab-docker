# Лабораторная работа: Динамический массив, Стек и Очередь на C++

## О проекте
Данная лабораторная работа представляет собой реализацию трёх фундаментальных структур данных на языке C++:
- Динамический массив с автоматическим расширением
- Стек на основе динамического массива
- Очередь, реализованная на двух стеках

## Структуры данных

### Динамический массив (DynamicArray)
Класс реализует динамический массив, который автоматически увеличивает свою ёмкость при необходимости. Поддерживаются следующие операции:
- Добавление элементов в начало и конец
- Вставка по индексу
- Удаление элементов
- Поиск минимального и максимального элемента
- Вычисление суммы всех элементов
- Перегрузка операторов [] и ()

### Стек (Stack)
Реализация стека на основе динамического массива с принципом LIFO (Last In, First Out):
- Добавление элемента на вершину (push)
- Извлечение элемента с вершины (pop)
- Просмотр верхнего элемента без извлечения (top)
- Проверка на пустоту и переполнение

### Очередь (Queue)
Реализация очереди на двух стеках с принципом FIFO (First In, First Out):
- Добавление элемента в конец очереди (enqueue)
- Извлечение элемента из начала очереди (dequeue)
- Просмотр первого элемента без извлечения (front)
- Эффективная реализация через два стека
