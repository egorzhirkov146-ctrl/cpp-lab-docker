# Лабораторная работа: Динамический массив, Стек и Очередь

## Запуск через Docker

1. Собрать образ:
   docker build -t lab .

2. Запустить программу:
   docker run --rm lab

### Структура проекта

- DynamicArray.cc / DynamicArray.h - динамический массив
- stack.cc / stack.h - стек на основе DynamicArray
- queue.cc / queue.h - очередь на двух стеках
- main.cc - тестовая программа
- Dockerfile - для создания контейнера

#### Пример вывода

=== Тестирование DynamicArray ===
Массив: 1.2 10.5 20.3 5.7 
Максимум: 20.3
Минимум: 1.2
Сумма: 37.7

=== Тестирование Stack ===
Добавлено: 10
Добавлено: 20
Добавлено: 30
Верхний элемент: 30
Извлекаем: 30

=== Тестирование Queue ===
Первый в очереди: 1.1
Извлекаем: 1.1
Теперь первый: 2.2
